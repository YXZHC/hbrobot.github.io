# 自主移动机器人使用说明文档

## 文档概述
本文档为专用自主移动机器人配套完整操作、硬件、软件、导航开发手册，面向高校课程实验、课堂实操演示、课程设计与课外科创实践人员编写。

文档基于 MkDocs 构建，支持在线网页浏览，源码托管于 GitHub，可随时迭代更新、协作修订。

## 文档内容板块
1. **快速上手**
    开箱验收、充电规范、开关机流程、急停操作，零基础快速启动机器人。
2. **硬件介绍**
    整机结构拆解、激光雷达、视觉相机、IMU、驱动轮组、电源及各类外接接口详细说明。
3. **软件与开发环境**
    车载系统架构、ROS/ROS2 环境部署、软件安装、固件升级、调试工具使用教程。
4. **基础操控**
    键盘、手柄手动遥控，程序启停，设备指示灯状态解读。
5. **自主导航功能**
    SLAM 建图、地图存储加载、路径规划、动态避障、导航点位设定实操步骤。
6. **安全规范**
    场地使用要求、安全操作距离、故障紧急处置方案，规避设备与人身风险。
7. **附录 FAQ**
    通信连接失败、导航漂移、传感器无数据、程序闪退等高频问题排查方案。

## 使用建议
- 初次使用请严格按照「快速上手」章节顺序操作，切勿跳过安全检查步骤；
- 课程实验可按需截取对应章节打印或导出 PDF 作为实验指导书；
- 文档支持多人协作修改，可 Fork 仓库提交修订建议。
